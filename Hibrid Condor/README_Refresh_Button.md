# Как включить кнопку обновления в Excel

1. Открой файл `Hybrid_Condor_v4_desktop.xlsx` в настольном Excel для Windows.
2. Сохрани копию как **Excel Macro-Enabled Workbook (.xlsm)**.
3. Нажми `Alt+F11` → `Insert` → `Module`.
4. Импортируй файл `RefreshQuotesAndCalc.bas` или вставь его код вручную.
5. Вернись в Excel → вкладка **Разработчик** → **Вставить** → **Кнопка (Form Control)**.
6. Нарисуй кнопку на листе `Dashbord` и назначь макрос `RefreshQuotesAndCalc`.
7. Для работы формул веб-запросов Excel должен поддерживать `WEBSERVICE` и `FILTERXML` на Windows Excel 2013+.[cite:35][cite:54]

## Что делает макрос
- выполняет `ThisWorkbook.RefreshAll`;
- пересчитывает формулы;
- показывает сообщение об успешном обновлении.[cite:53][cite:56]

## Важное ограничение
При программной генерации `.xlsm` библиотеки часто ломают или теряют связанные кнопки, поэтому безопаснее передавать макрос отдельно и подключать его уже в Excel.[cite:49][cite:50][cite:51]
